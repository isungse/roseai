인사평가 안내문

올해 처음 시행되는 전자인사평가 시스템 운영과 관련하여, 평가자분들께서 반드시 준수하셔야 할 기준을 안내드립니다. 시스템 특성상 점수 미충족 시 평가 완료가 되지 않으니 아래 내용을 필히 확인해 주시기 바랍니다.

⚠️평가 시 유의사항

평가 완료 후에는 시스템 특성상 수정 및 재평가가 어려우므로 최종 제출 전 다시 한번 확인 바랍니다. 처음 도입되는 시스템인 만큼 다소 번거로우시더라도 원활한 인사 평가를 위해 기한 내 완료를 부탁드립니다.

🛠️시스템 오류 및 기술 지원 문의

로그인 오류(생년월일 불일치), 평가 창 초기화 등 시스템 관련 문제는 아래 연락처로 문의해 주시면 순차적으로 피드백 드리겠습니다.

인사평가 시행방법

구분평가 대상 (평가자 → 피평가자)평가 방법
1. 직원평가부서 내 동료 간 상호평가[직원평가] 클릭 → 소속 그룹 및 부서 선택 → 평가자 본인 성명 선택 → 생년월일 입력
2. 팀원평가팀장 → 부서원 평가[팀원평가] 클릭 → 소속 그룹 및 부서 선택 → 평가자 본인 성명 선택 → 생년월일 입력
3. 팀장평가팀장 간 상호평가[팀장평가] 클릭 → 소속 그룹 및 부서 선택 → 평가자 본인 성명 선택 → 생년월일 입력
4. 임원평가임원 → 담당 팀장 평가[임원평가] 클릭 → 평가자 본인 성명 선택 → 비밀번호 입력 후 진행
  • ※ 임원평가 해당자(할당된 비밀번호는 개별전달) : 병원장, 행정부원장, 간호부원장, 간호부장, 검진부장
  • ※ 본인 인증 확인 : 1~3번 평가 시 입력하는 생년월일(6자리)이 정확하지 않으면 평가 창이 활성화되지 않으니 주의 바랍니다. (맞는 생년월일을 입력하여도 접속이 불가할 경우 인사총무팀으로 연락 바랍니다.)
  • ※ 평가 그룹 선택 : 본인이 속한 그룹(예: 간호부, 행정/의료기사, 검진센터 등)을 먼저 정확히 확인한 후 평가를 시작해 주세요.
  • ※ 저장 및 제출 : 각 대상자에 대한 평가를 마친 후 반드시 오른쪽 하단 '평가 제출하기' 버튼을 눌러야 하며, 최종 완료 후에는 수정이 어려울 수 있으니 신중히 기입해 주시기 바랍니다.
